1. **Download Debian 12 ISO:**
   - Go to the official Debian website (https://www.debian.org) and navigate to the downloads page.
   - Choose the appropriate installation image for your system architecture (e.g., 64-bit or 32-bit) and download the ISO file.

2. **Create a bootable USB or DVD:**
   - Insert a blank USB drive or DVD into your computer.
   - Use a tool like Rufus (for Windows) or Etcher (for macOS and Linux) to create a bootable USB drive or burn the ISO file to a DVD.

3. **Boot from the installation media:**
   - Restart your computer.
   - Enter the BIOS or UEFI settings by pressing the designated key during startup (e.g., F2, Del, Esc).
   - In the BIOS/UEFI settings, change the boot order to prioritize the USB drive or DVD.
   - Save the changes and exit the BIOS/UEFI settings.

4. **Start the Debian installation:**
   - The system will boot from the installation media, and you'll see the Debian installer screen.
   - Select the "Install" option to begin the installation process.
